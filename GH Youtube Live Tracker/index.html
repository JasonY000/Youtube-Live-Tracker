Add channel ID to track

Option #1: Channel id manual input

Go to the target youtube channel main page, right click and select "view page source", look for "channel_id=" and copy the following charaters.

It will look somthing like this "channel_id=UCvzPLP9oQwU--Y4r9id_unA" and you would want to copy "UCvzPLP9oQwU--Y4r9id_unA"

Option #2: Channel name lookUp

Type in channel name and click "search", click "view" to go to channel page, press "add" to track
